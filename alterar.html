O que são Dashboards?

O Dashboard é uma ferramenta poderosa para criar painéis interativos, combinando mapas, gráficos, tabelas, indicadores e filtros em uma única interface visual. Ele permite que usuários finais explorem dados geoespaciais de forma intuitiva, facilitando a análise de dados e a tomada de decisão.

Antes de começar, você precisa:

  • Estar autenticado no Geohab com permissões para criar conteúdo;
  • Ter ao menos um mapa ou camada configurado;
  • Ter acesso a uma ou mais fontes de dados (como camadas vetoriais com atributos).
Passo a Passo: Criando um Dashboard
1. Acessando a Área de Dashboards

1 Acesse o ícone "Apps" do menu.

2 Em seguida, clique no segundo ícone (novo "Dashboard").

×
 
Configurações Iniciais

Preencha os campos básicos:

  • Título: Nome do painel.
  • Resumo (opcional): Breve descrição.
  • Tags (opcional): Tags para facilitar a busca.
  • Compartilhamento: Defina quem pode visualizar ou editar (público, privado, por grupos).
    • Clique em Criar.
Tipos de Widget do Dashboard

O dashboard possui uma interface modular composta por linhas e colunas. Você pode adicionar Widgets em qualquer posição. Os principais componentes são:

1 Gráficos de barras / pizza / linha.

2 Tabela de atributos.

3 Texto (para títulos ou explicações).

4 Indicadores (KPIs).

5 Mapa: Permite inserir mapas interativos (pré-existentes ou criados do zero). É possível ativar a seleção de feições ou o uso de filtros no mapa.

×
 

Para adicionar um widget:

1 Clique em Adicionar Widget.

2 Escolha o tipo (gráfico, tabela, etc.).

3 Selecione a camada de dados.

4 Defina as métricas (ex: contagem, soma, média).

5 Configure os filtros e agrupamentos.

6 Clique em "Salvar".

×
 
4. Filtros Interativos

Os dashboards podem conter filtros globais aplicados a todos os widgets:

  • Clique em Adicionar Filtro.
  • Escolha a camada e o campo de atributo.
  • Os filtros aparecem como menus suspensos, caixas de seleção ou campos de texto.
  • Quando aplicados, os widgets são atualizados automaticamente.
×
 
5. Layout e Design

Você pode ajustar o layout:

  • Redimensione os widgets com arrastar e soltar.
  • Organize por seções lógicas.
  • Use widgets de texto para títulos, legendas ou explicações.
×
 
6. Salvando e Compartilhando

Após concluir, clique em Salvar.

Para compartilhar:

  • Use o botão Compartilhar para copiar o link.
  • Controle quem pode visualizar/editar.

Dicas Avançadas
  • Use expressões de agregação para KPIs (ex: média de população por região).
  • Combine mapas com widgets vinculados para análises geoespaciais interativas.
  • Habilite o modo de seleção no mapa para filtrar dados clicando em feições.
Exemplo Prático

Imagine que você tem uma camada de incidência de dengue por município. Um dashboard pode conter:

  • Mapa com os municípios e os casos.
  • Gráfico de barras: Casos por mês.
  • Indicador: Total de casos no ano.
  • Filtro: Ano ou estado.
  • Tabela: Lista de municípios com mais de 100 casos.
Boas Práticas
  • Mantenha o layout limpo e objetivo.
  • Nomeie widgets e filtros claramente.
  • Evite muitos gráficos em um único painel – prefira vários dashboards pequenos.
  • Revise os metadados das camadas para garantir que os campos estão bem definidos.

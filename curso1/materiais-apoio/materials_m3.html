Como Encontrar Dados no Catálogo


O Catálogo (GeoNetwork) é a porta de entrada para encontrar e entender os dados geoespaciais disponíveis. Siga os passos abaixo para localizar as camadas que você precisa.

  • 1 Acesse o Catálogo

Na página inicial do Geohab, a interface do Catálogo 1 já é apresentada. Se estiver em outra seção, clique no menu superior para retornar à interface de busca de metadados (dados sobre os dados: informações descritivas de uma camada).

  • 2 Realize a Busca

1 Use a caixa de busca principal para encontrar camadas por nome ou palavras-chave. Por exemplo: "hidrografia", "saneamento" ou "loteamentos".

2 Clique na lupa ou pressione Enter ↵ no teclado para ver os resultados. Você também pode encontrar camadas em destaque rolando a página.

3 Em News feed temos os metadados publicados mais recentes.

  • 2 Prévia da Camada: Miniatura do mapa.
  • 3 Nome da Camada: Título e link.
  • 4 Resumo: Descrição da camada.

Visualizando Camadas


O Visualizador é um dos componentes principais do Geohab, integrado ao Serviços (GeoServer) e ao Catálogo (GeoNetwork). Ele é a principal ferramenta para a exploração visual interativa das camadas geoespaciais.

Este guia demonstra as formas mais comuns de adicionar e visualizar uma camada, permitindo que você analise se ela atende aos seus requisitos.

Nota Importante: Lembre-se que para visualizar efetivamente uma camada, seu perfil de usuário no Geohab deve ter as permissões de acesso necessárias para o serviço (WMS/WFS) correspondente.

1. A partir do Catálogo

Utilize o Catálogo (GeoNetwork) para encontrar a camada de seu interesse.

1 Na página de detalhes do metadado da camada desejada, selecione o tipo de GeoServiço (WMS para mapa ou WFS para vetor) e clique em visualizar.

- A camada será automaticamente carregada no Visualizador.

Filtrando Dados em Camadas Espaciais


O Visualizador, componente de visualização do Geohab, oferece recursos avançados para filtrar dados em camadas espaciais. Esta funcionalidade permite refinar a visualização de acordo com critérios específicos, facilitando a análise de informações relevantes.

O processo de filtragem pode ser aplicado a diferentes tipos de camadas, permitindo selecionar elementos com base em atributos tabulares ou condições espaciais.

Nota Importante: Para utilizar os recursos de filtragem, seu perfil de usuário no Geohab deve ter as permissões de acesso necessárias para a camada que deseja filtrar.

Processo de Filtragem no Visualizador

  • 1 Acessar a Camada: No Visualizador, selecione a camada que deseja filtrar no painel de camadas. Clique com o botão direito sobre a camada ou no ícone de opções para acessar o menu de ferramentas da camada.
  • 2 Abrir a Ferramenta de Filtro: No menu de opções da camada, selecione a ferramenta Filtro ou Filter. Isso abrirá uma interface onde você pode definir os critérios de filtragem com base nos atributos disponíveis na camada.
  • 3 Definir Critérios de Filtragem: Na interface de filtro, você pode:
    • Selecionar o atributo que deseja filtrar
    • Escolher o operador de comparação (igual a, diferente de, maior que, contém, etc.)
    • Inserir o valor ou valores para a condição
    • Combinar múltiplas condições usando operadores lógicos (E, OU)
  • 4 Aplicar e Visualizar o Resultado: Após definir os critérios, clique em Aplicar ou Apply. O mapa será atualizado para mostrar apenas os elementos que atendem aos critérios de filtragem definidos. Você pode salvar o filtro para uso futuro ou compartilhá-lo com outros usuários.

Filtro Espacial em Camadas


O Visualizador oferece poderosas funcionalidades de filtro espacial que permitem selecionar elementos geográficos com base em sua localização e relação espacial com outros elementos.

Os filtros espaciais são ferramentas essenciais para análise geográfica, permitindo isolar feições específicas com base em critérios como proximidade, intersecção, contenção e outras relações topológicas.

Nota Importante: Para utilizar os recursos de filtro espacial, além das permissões de acesso à camada, é necessário que a camada possua geometrias adequadas para este tipo de operação.

Processo de Filtro Espacial no Visualizador

  • 1 Acessar a Ferramenta de Filtro Espacial: No Visualizador, selecione a camada que deseja filtrar no painel de camadas. Clique com o botão direito sobre a camada e selecione a opção Filtro Espacial ou Spatial Filter.
  • 2 Definir a Geometria de Referência: Você pode definir a geometria de referência para o filtro espacial de várias formas:
    • Desenhando diretamente no mapa (ponto, linha ou polígono)
    • Selecionando uma feição de outra camada
    • Importando uma geometria de um arquivo
  • 3 Selecionar o Tipo de Relação Espacial: Escolha o tipo de relação espacial que deseja aplicar:
    • Intersect - Seleciona elementos que interceptam a geometria de referência
    • Contains - Seleciona elementos contidos na geometria de referência
    • Within - Seleciona elementos que contêm a geometria de referência
    • Touches - Seleciona elementos que tocam a geometria de referência
    • DWithin - Seleciona elementos dentro de uma distância específica
  • 4 Aplicar e Analisar o Resultado: Após definir a geometria e o tipo de relação, clique em Aplicar ou Apply. O mapa será atualizado para mostrar apenas os elementos que atendem ao critério espacial definido. Os resultados podem ser exportados ou combinados com outros filtros para análises mais complexas.

Banco de Dados

Gerenciando e acessando dados geoespaciais.

Este módulo explora as diferentes formas como os dados geoespaciais são armazenados e acessados dentro do Geohab. Você aprenderá sobre as conexões com bancos de dados robustos como PostgreSQL (com PostGIS) e MS SQL Server, além de entender o processo de incorporação e utilização de dados provenientes de fontes externas.

PÚBLICO-ALVO

Perfis de acesso: Editor, Revisor, Administrador Local e Super Usuário.

PRÉ-REQUISITOS

Nenhum conhecimento prévio é necessário.

Objetivos de Aprendizagem:

  1. Entender o processo de conexão com bancos de dados PostgreSQL (PostGIS).
  2. Entender o processo de conexão com bancos de dados MS SQL Server.
  3. Aprender o processo de carga de dados para o banco de dados.
  4. Entender quando, como e onde armazenar dados geográficos.
